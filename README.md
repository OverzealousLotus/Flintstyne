## Flintstyne
Flintstyne is a personal KubeJS package. There are various fixes, tweaks, and compatibility additions.

### Compatibility
- Tinkers' Construct smelting for SimplySwords
- Biomes 'O Plenty flowers have a custom tag added to extract latex for Thermal.
- Tags for Pams'/Thermal/Farmers recipes.
- Molten Mochite & Casts for Glintstyne

### Fixes
- Removes vanilla recipes burdening JEI windows when tagged alternatives present.

### Additions
- Overworld Ancient Debris Ore

### Texture Rips
- [Ancient Debris Ore & Shard textures](https://www.curseforge.com/minecraft/mc-mods/skays-overworld-netherite-ore)


#### Footnote
Flintstyne is meant to pair well with Glintstyne, or any other mod which fixes Mojank's lava issue!
